# gift-idea-generator
Webapp to generate ideas to help with gift shopping

## Motivation
Finding the perfect gift can be surprisingly difficult, People rarely say exactly what they want. This app was created to simplify that process by helping users discover thoughtful, personalized gift ideas based on the recipient’s interests and preferences.
